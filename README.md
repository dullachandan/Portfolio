# Portfolio

This repository contains a personal portfolio website built with HTML, CSS, and JavaScript.

## Contents

- `index.html` — main portfolio page
- `style.css` — visual styles
- `script.js` — interactive behavior
- `hero-sketch-male.png` — hero image used on the site

## Notes

Use a simple static hosting service such as GitHub Pages to publish this portfolio.

## Deployment

To preview locally, open `index.html` in a browser or use a local server.
